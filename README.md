# 🎙️ Real-Time Voice Assistant

This is a real-time voice assistant that captures speech from your microphone, converts it into text, generates a smart reply, and then speaks the response back to you.

---

## 🚀 Features

- 🎧 Real-time voice input
- ✍️ Speech-to-text conversion
- 🧠 AI-based conversation
- 🔊 Text-to-speech audio response
- 🔁 Looping dialogue system
